Project Overview

This project presents a comprehensive SQL-based analysis of donor and donation data to identify trends in donor behaviour, demographic characteristics, geographic distribution, and fundraising performance. The analysis was conducted as part of a SQL Capstone Project to demonstrate practical SQL skills, business problem-solving, and data-driven decision-making.
The project focuses on transforming raw donor and donation data into meaningful business insights that support fundraising strategy, donor segmentation, and organisational planning.

Business Problem

Non-profit organisations rely on donations to sustain their operations. However, without analysing donor data, it is difficult to understand:
Which donor groups contribute the most.
Which regions generate the highest donation revenue.
Which occupations are associated with higher donations.
How donation frequency influences fundraising performance.
Opportunities for improving fundraising campaigns.
This project addresses these challenges by analysing donor and donation data to produce actionable business insights and recommendations.

Project Objectives

The objectives of this project were to:
Assess the overall donation performance.
Analyse donor demographics and donation behaviour.
Identify geographic donation trends.
Evaluate donor segmentation by occupation.
Examine high-value donation patterns.
Generate business recommendations to improve fundraising effectiveness.

Key Findings

Male donors contributed a slightly higher total donation amount than female donors.
California generated the highest total donation revenue among all states analysed.
Human Resources professionals recorded the highest overall donation amount.
High-value donations (greater than $200) accounted for a significant proportion of fundraising revenue.
Ford was the most common vehicle brand among the highest-value donors.

Recommendations

Develop targeted fundraising campaigns for high-performing states such as California and Texas to maximise donation revenue.
Strengthen donor engagement programmes for high-value donors to improve donor retention and lifetime value.
Implement occupation-based fundraising strategies by focusing on sectors with consistently higher donation contributions.
Encourage recurring donations through personalised communication and donor recognition initiatives.
Use donor demographic insights to improve fundraising campaign segmentation and effectiveness.

How to Reproduce the Analysis

1. Clone this repository.
2. Open PostgreSQL and pgAdmin 4.
3. Create a new PostgreSQL database.
4. Import the donor and donation datasets.
5. Execute the SQL scripts in the sql folder sequentially.
6. Review the analytical portfolio and visualisations for business insights.
